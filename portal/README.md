# Portal (Página Central)

Esta pasta contém a página inicial para orientar o usuário ao formulário correto:

- **Empresas**: Formulário de **Demanda**
- **Instituições**: Formulário de **Oferta**

## Desenvolvimento (local)

1. Suba os formulários:
   - Demanda: `npm run dev -- --port 5173` em `frontend-demanda/`
   - Oferta: `npm run dev -- --port 5174` em `frontend-oferta/`
2. Abra `portal/index.html` no navegador.

Se você publicar os formulários em URLs diferentes, atualize os links no `portal/index.html`.
